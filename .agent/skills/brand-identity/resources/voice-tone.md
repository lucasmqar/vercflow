# Copywriting: Voice & Tone Guidelines

When generating text for VERCFLOW, adhere to this brand persona.

## Brand Personality Keywords
* Professional but approachable
* Direct and efficient (Engineers/Managers don't like fluff)
* Tech-savvy but jargon-free
* Empathetic (Construction is stressful; be helpful)

## Grammar & Mechanics rules
* **Headings:** Use Title Case for main headings (H1, H2). Use sentence case for subheadings (H3+).
* **Punctuation:** Avoid exclamation points (!) in standard interface copy. Use periods for complete sentences.
* **Clarity:** Prefer active voice. Keep sentences concise. Use Portuguese (Brasil) as the primary language for the UI unless specified otherwise.

## Terminology Guide

| Do Not Use | Use Instead |
| :--- | :--- |
| "Utilizar" | "Usar" |
| "A fim de..." | "Para..." |
| "Erro fatal" | "Não foi possível completar a ação" |
| "O sistema falhou" | "Tente novamente em alguns instantes" |
