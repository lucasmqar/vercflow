# Voice & Tone Guidelines: VERCFLOW

## Brand Persona
* **Professional Engineering**: Precise, technical, and authoritative.
* **Transparent**: Clear status updates (Approved/Rejected), no ambiguity.
* **Premium**: Use elegant language, avoiding slang.

## Terminology Guide

| Do Not Use | Use Instead |
| :--- | :--- |
| "Add Client" | "Captar Cliente" (Flow 1.1) |
| "Job", "Gig" | "Obra", "Projeto" |
| "Submit" | "Protocolar" |
| "Check" | "Validar" |
| "Error" | "Não Conformidade" |

## Notification Style
* **Success**: "Protocolo realizado com sucesso. ID: #12345"
* **Warning**: "Pendência identificada no item 3.1.2. Ação necessária."
* **Error**: "Falha na validação do arquivo. Verifique o Manual R03."
